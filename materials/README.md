# 자료 보관 가이드

이 폴더는 공식 자료 또는 그에 기반한 개인 노트를 보관할 때 참고하는 안내용 공간이다. 실제 사실 기준은 `docs/01_verified_sources.md`를 따른다.

## 여기에 두어도 되는 것
- ETS 또는 한국TOEIC위원회에서 직접 받은 공식 PDF
- 공식 자료를 바탕으로 만든 개인 요약
- 필요 시 `supplemental`로 명시한 보조 모의고사 1권

## 함부로 두면 안 되는 것
- 보호되는 TOEIC 문제 전체 복사본
- 대량 정답표 복사본
- 사용자가 직접 제공하지 않은 전체 스크립트 복사본

## 이름 규칙
- 공식 자료: `official_`로 시작
- 보조 자료: `supplemental_`로 시작
- 개인 노트: `note_`로 시작

## 사용 규칙
- 이 폴더 안 문서가 사실을 말할 때는 공식 링크를 직접 달거나 `docs/01_verified_sources.md`를 참조한다.
