# 어휘·표현 운영 인덱스

- 용도: 코어 어휘 구조의 진입점
- 대상 파트: 코어 어휘, LC/RC 패러프레이즈
- 입력 자료: 마스터 목록, 주제 세트, 어휘 복습 큐
- 사후 평가 여부: 있음. 추천 모델과 반복 혼동 누적에 반영

## 사용 순서
1. 전체 풀 확인: `master_core_200.md`
2. 주제별 사용: `topic_sets/` 아래 해당 주제 문서
3. 불안정 항목 이동: `review_queue.md`
4. 부족 시 보충 생성: `templates/synthetic_vocab_expression_set_template.md`

## 주제 세트
- 일정, 회의, 약속: `topic_sets/topic_schedule_meeting.md`
- 이메일, 사무, 커뮤니케이션: `topic_sets/topic_office_communication.md`
- 채용, 인사, 인터뷰: `topic_sets/topic_hiring_hr.md`
- 주문, 배송, 청구: `topic_sets/topic_order_shipping.md`
- 예약, 여행, 행사: `topic_sets/topic_travel_hotel.md`
- 회의, 공지, 보고: `topic_sets/topic_notice_report.md`
- 유지보수, 고장, 수리: `topic_sets/topic_maintenance_tech.md`
- 고객문의, 환불, 불만: `topic_sets/topic_customer_service.md`
- 계약, 마케팅, 재무: `topic_sets/topic_contract_marketing_finance.md`
