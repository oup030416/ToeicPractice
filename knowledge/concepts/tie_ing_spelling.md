# 개념 카드

- 개념: tie 진행형 철자
- 핵심 규칙: 동사가 `-ie`로 끝나면 진행형에서 `ie`를 `y`로 바꾸고 `-ing`를 붙인다.
- 자주 틀리는 이유: 원형을 그대로 유지해서 `tieing`처럼 적기 쉽다.
- 관련 파트: grammar
- 영어 예문: tie -> tying, die -> dying, lie -> lying
- 출처/앵커: tracking/qna_master_log.md
- 마지막 갱신일: 2026-03-13
- 질문 횟수: 1

## 빠른 설명
- `tie`의 진행형은 `tying`이 맞다.
- 같은 규칙으로 `die -> dying`, `lie -> lying`이 된다.

## 헷갈리는 포인트
- `tieing`은 표준 진행형 철자가 아니다.
- `-ie`가 아닌 일반 동사에는 이 규칙을 그대로 적용하지 않는다.
