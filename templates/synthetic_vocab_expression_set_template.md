# 보충용 Synthetic Vocab Set 템플릿

- 용도: 코어 어휘·표현 부족 시 제한 보강 세트 생성
- 대상 파트: 코어 어휘, Part 2 패러프레이즈, Part 7 문맥 보강
- 입력 자료: 공식 자료에서 확인된 직장 영어 맥락, 승인된 마스터 목록, 약점 주제
- 사후 평가 여부: 있음. `toeic-review-auditor` 통과 전 실사용 금지

## 생성 규칙
- 이 문서는 반드시 `보충용 synthetic vocab/expression set`라고 표시한다.
- 공식 문항 문장, 스크립트, 보기, 정답 배열을 그대로 복제하지 않는다.
- full mock test 형식으로 만들지 않는다.
- 허용 범위는 어휘-문맥 매칭, 표현 묶음 재구성, 패러프레이즈 대응, 제한 빈칸 반응형이다.
