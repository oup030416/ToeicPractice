# 코어 어휘 마스터 목록 템플릿

- 용도: 프로젝트 전체 코어 어휘·표현의 기준 목록 관리
- 대상 파트: 코어 어휘, Part 2, Part 3/4, Part 7
- 입력 자료: 사용자 준비 목록, 공식 자료 기반 표현 수집, 승인된 보충 세트
- 사후 평가 여부: 있음. 주제 세트 참조 빈도와 복습 이동 여부 점검

## 운영 규칙
- 이 문서는 전체 풀만 관리한다.
- 실제 사용은 주제 세트와 어휘 복습 큐를 우선한다.
- 보충 생성물은 `보충용 synthetic vocab/expression set` 표시가 있을 때만 편입 후보로 본다.
