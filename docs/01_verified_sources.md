# 검증된 출처

검증 기준일: 2026-03-12 (Asia/Seoul)

이 문서는 이 워크스페이스에서 사실 판단의 기준이 되는 출처 목록이다. 시험 형식, 점수 해석, 자료 현황, 저작권 처리와 관련된 주장은 아래 출처를 먼저 따른다.

## 출처 우선순위
1. ETS 공식 TOEIC 페이지와 PDF
2. 한국TOEIC위원회 공식 페이지
3. 사용자가 확보한 공식 자료
4. 명시적으로 `supplemental` 표시한 보조 자료

## 프로젝트 적용 원칙
- 핵심 코퍼스는 한국TOEIC위원회 공개문항과 ETS 공식 자료로 고정한다.
- 원점수 몇 개를 특정 점수로 단정하지 않는다.
- 한 번의 모의고사 결과만으로 정확한 예상 점수를 말하지 않는다.
- 보호 자료는 복사보다 링크와 문항 메타데이터 중심으로 다룬다.
- 학습자 상태 해석은 ETS 설명 자료와 사용자 보고를 결합한 코칭 해석으로 명시한다.
