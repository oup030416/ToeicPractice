# 적응형 추천 모델

이 문서는 현재 프로젝트에서 약점 우선순위를 계산하고 다음 학습 제안을 만드는 기준 문서다.

## 입력 단위
- 집계 단위: `파트 + 하위능력 + 주제`
- 예시:
  - `Part 2 + implied meaning + indirect response`
  - `Part 5 + grammar + parts of speech`

## 입력값
- 최근 시도 수
- 최근 정답률
- 최근 오답 수
- 반복 혼동 횟수
- 최근 Q&A 질문 횟수
- 시간 압박 플래그
- 마지막 노출일

## priority score 계산
- 점수 범위: 0-100
- 기본 계산식:
  - 정확도 부족 점수 = `(100 - 최근 정답률) * 0.35`
  - 오답 빈도 점수 = `min(최근 오답 수 * 5, 100) * 0.25`
  - 반복 혼동 점수:
    - 0회 = `0 * 0.20`
    - 1회 = `40 * 0.20`
    - 2회 = `70 * 0.20`
    - 3회 이상 = `100 * 0.20`
  - 최근성 점수:
    - 마지막 노출이 3일 이내이며 unresolved = `100 * 0.10`
    - 4-7일 = `70 * 0.10`
    - 8-10일 = `40 * 0.10`
    - 11일 이상 = `20 * 0.10`
  - 시간 압박 점수:
    - 없음 = `0 * 0.10`
    - 약함 = `50 * 0.10`
    - 강함 = `100 * 0.10`

## 강제 상향 규칙
- 반복 혼동이 3회 이상이면 `우선 보강 후보`로 강제 표기한다.
- 최근 Q&A 질문이 3회 이상 누적되면 priority score와 별개로 `반복 개념 경고`를 붙인다.
- 같은 주제에서 최근 시도 2회 연속 정답률 50% 미만이면 `즉시 보강` 후보로 올린다.

## 추천 출력 규칙
추천은 항상 아래 3단계로 정리한다.
1. `지금 가장 먼저 보강할 것`
2. `다음 세션에서 다시 볼 것`
3. `당장 보류해도 되는 것`

## 추천 필드
- 무엇을
- 왜
- 근거 데이터
- 추천 강도

## 추천 종류
- 약한 파트 재노출
- 특정 하위능력 보강
- 특정 어휘 주제 복습
- 특정 개념 카드 재확인
- 드릴 생성 요청

## 추천 강도 기준
- `critical`: priority score 75 이상, 또는 강제 상향 규칙 충족
- `high`: priority score 60-74
- `medium`: priority score 40-59
- `low`: priority score 0-39
