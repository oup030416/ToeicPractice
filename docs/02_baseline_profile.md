# 현재 실력 프로필

이 문서는 사용자가 설명한 현재 상태를 ETS 공식 점수 설명 자료와 연결해 해석한 문서다.

## 상태 표기 원칙
- 이 문서는 공식 성적표가 아니다.
- 이 문서는 코칭 해석이다.
- 근거 출처:
  - [ETS TOEIC LR 점수 설명 자료 PDF](https://www.ets.org/pdfs/toeic/toeic-listening-reading-score-descriptors.pdf)
  - 사용자 현재 상태 설명

## Listening: 200-300대 성향 해석

### 공식 설명과의 연결
- LC 200대 설명은 쉬운 어휘와 직접적인 정보에는 일부 반응하지만, 패러프레이즈나 긴 지문 중간 정보, 부정 표현, 다소 어려운 어휘에서 흔들릴 수 있음을 보여 준다.
- LC 300대 설명은 중심 내용이나 목적을 어느 정도 잡을 수 있지만, 간접응답, 예측이 어려운 응답, 긴 지문 내부의 연결 정보에서는 여전히 약할 수 있음을 보여 준다.

### 현재 상태 해석
- 쉬운 어휘와 반복이 있으면 중심 내용과 일부 세부를 잡는다는 현재 진술은 LC 200-300대 설명과 자연스럽게 맞닿는다.
- 간접응답, 패러프레이즈, 긴 지문 중간 정보가 약하다는 진술은 Part 2와 Part 3/4의 핵심 병목을 직접 가리킨다.

## Reading: 150-250대 성향 해석

### 공식 설명과의 연결
- RC 150대 설명은 단어가 직접 맞아떨어지는 쉬운 사실 정보와 기본 문법은 일부 처리하지만, 추론과 패러프레이즈, 연결 판단은 약할 수 있음을 보여 준다.
- RC 250대 설명은 제한된 범위에서 단순 추론과 일부 연결은 가능하지만, 두 문장 이상 연결, 패러프레이즈 기반 판단, 어려운 문법과 어휘에서는 약할 수 있음을 보여 준다.

### 현재 상태 해석
- 쉬운 어휘와 기본 문법은 일부 처리 가능하지만, 패러프레이즈와 문장 연결, 복합 문법이 약하다는 설명은 RC 150-250대 성향과 잘 맞는다.
- 따라서 읽기에서는 이론 암기보다 속도와 구조 판단을 먼저 끌어올리는 편이 더 효율적이다.

## 코칭 결론
- 가장 먼저 반복 추적해야 할 Listening 병목: Part 2 간접응답과 Part 3/4 중간 연결 정보
- 가장 먼저 반복 추적해야 할 Reading 병목: Part 5/6 문법 반응과 Part 7 패러프레이즈·근거 위치
- 추천 모델은 위 약점을 기본 초기값으로 쓰되, 실제 시도 기록이 쌓이면 그 결과로 우선순위를 덮어쓴다.

## 이 문서가 뜻하지 않는 것
- 실제 성적이 이 범위에 고정된다는 뜻이 아니다.
- 모의고사 한 번으로 정확한 환산 점수를 알 수 있다는 뜻이 아니다.
- 점수 추정을 이 프로젝트의 주요 출력으로 삼겠다는 뜻이 아니다.
